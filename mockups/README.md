# Mockups and research assets

This folder holds **Android Auto / Media3** research notes, **Figma** references, **exported PNGs**, and a **Svelte** browser mockup (migrated from the former **`pa2-car-plugin`** umbrella repo).

| Path | Purpose |
|------|---------|
| **[research/](research/)** | Notes and link lists (e.g. Figma URLs) |
| **[assets/](assets/)** | Static images — hero and DHU reference screenshots for docs and README |
| **[web-mockup/](web-mockup/)** | Svelte + Vite mockup (`npm ci && npm run build`); not deployed by default from this branch |

**Full UX research, design system, and user stories** are under **[docs/](../docs/README.md)** on this branch.

**GitHub Pages:** research Markdown is built with MkDocs — see **`mkdocs.yml`** and **`.github/workflows/pages.yml`** in the repo root.

**Implementation** (Kotlin, Gradle) is on the **`main`** branch — not here.
