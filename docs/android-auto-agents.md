# Android Auto agent pack (scripts + MCP stubs)

This branch includes **`android-auto-agents/`** at the **repository root** (next to `docs/`). It is **not** part of the MkDocs `docs/` tree, so the static site links out to the folder on GitHub.

**Contents:** bash scripts (DHU, Gradle plugin template, adb install, logcat, mockup verify), **workflows**, **role** markdown files, and **MCP tool descriptor** JSON under `mcp/tool-descriptors/` (stubs for a future MCP server — no server is implemented in-repo).

- **Browse on GitHub:** [android-auto-agents/README.md](https://github.com/shahzebqazi/PowerAmpache2PluginTemplate/blob/mockups/android-auto-agents/README.md)

When working locally, open **`android-auto-agents/README.md`** from the repo root.
